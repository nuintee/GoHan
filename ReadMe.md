# GoHan

シンプルな UI/UX のランダムご飯選択アプリです。
ユーザーの位置情報に基づいて半径 3km 以内の飲食店が表示される仕様です。
今後のアップデートで時間、値段、国籍などの条件を付けて検索できる機能を追加予定です。
